# Triangle Mastercard - Customer Churn Prediction

A comprehensive data analytics project for predicting customer churn in Triangle Mastercard (Canadian Tire Financial Services) using synthetic transaction data and machine learning.

## Project Overview

This project generates realistic customer data with intentional data quality issues to practice:
- Data cleaning and preprocessing
- Null value imputation
- Hypothesis testing
- Logistic regression modeling
- Customer churn prediction

## Dataset

### Tables Generated

| Table | Rows | Description |
|-------|------|-------------|
| **raw_customers** | 1,000 | Customer demographics (age, province, join date) |
| **raw_subscriptions** | 1,000 | Card tier info (Member, Triangle, World Elite) |
| **raw_payments** | 5,050 | Transaction history with duplicates |
| **raw_marketing** | 3,000 | Campaign engagement data |
